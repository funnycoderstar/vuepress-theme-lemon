# vuepress-theme-lemon
A simple vuepress theme

# 使用
```js
npm i -D vuepress-theme-lemon
```
# 快速生成模板文件
```js
$ npm i -g miko-cli
$ miko-cli create myBlog vuepress-lemon
$ cd myBlog
$ npm run dev
```
